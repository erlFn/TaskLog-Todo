# Authentication Layout

- [] Authentication Layout (`Login Page` and `Register Page`) should have the same layout, but has different page rendered

# App Layout
- [] There should be two(2) `App Layout`. One is for user that has a role of admin, and the other one is for a user that has a role of a user.
- [] Each respective layout should contain a `Navigation Menu` that whatever page is rendered, the navigation menu is still shown.
